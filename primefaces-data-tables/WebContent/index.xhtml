PrimeFaces Data Tables



The following examples illustrate some of the p:dataTable options that are explained in the tutorial.

  • Basics. The var and value attributes of p:dataTable, and the nested p:column element.
  • Headers. headerText for columns, f:facet with name="header" for overall table.
  • Styling. Most of the time, you want to use the styles from the current theme for the table as a whole, but you often want to style various rows differently.
  • Resizable columns. Designating resizableColumns="true" lets the end user drag to shrink or expand the column width.
  • Sorting. sortBy and optionally sortFunction.
  • Pagination. Using paginator="true" and rows="n" to show only a subset of the rows and automatically get controls to move forward and backward in the data.
